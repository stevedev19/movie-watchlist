# Movie Watchlist

A responsive Movie Watchlist web app built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

- Add movies with title (required), year, and genre (optional)
- Organize movies into "To Watch" and "Watched" lists
- Mark movies as watched/unwatched
- Delete movies
- Filter movies by title (case-insensitive)
- Persistent storage using localStorage
- Responsive design with Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
